#bk0011m-desolate Desolate game BK-0011M version

Adaptation of the UKNC version for the BK-0011M soviet home/educational computer. Since both machines are compatible with
DEC PDP-11/03 and feature a pixel-addressable graphical display, adaptation was fairly trivial.
You will need either a real hardware, or any reliable emulator.


"Desolate", the game for Elektronika BK-0011M, the Soviet home/educational
personal computer.

Originally written for the TI-83 Plus programmable calculator by
Patrick Prendergast (tr1p1ea).

Ported to UKNC Soviet Educational computer by Nikita Zeemin (nzeemin), 2020
Adapted for BK-0011M by Alexander "Sandro" Tishin, 2021

Game is provided as the bootable floppy disk image (desolate01.img);
simply boot from it in any compatible emulator.


Controls to use in the game:
 - Arrows - movement
 - Space - look/shoot
 - <kbd>I</kbd> - Inventory
 - <kbd>Q</kbd> - switch look/shoot mode
 - <kbd>P</kbd> - return to menu
 - Other keys (usually I prefer <kbd>Tab</kbd>) - close all pop-ups
